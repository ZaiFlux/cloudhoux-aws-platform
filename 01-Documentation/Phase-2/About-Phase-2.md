# Frontend and Backend application development

## Phase 2 – Application Development

| Task | Description |
|------|-------------|
| **Develop the Frontend Interface** | Create the user interface (Login, Register, Dashboard, Upload Page, and File List) using **HTML, CSS, and JavaScript**. |
| **Develop the Backend API** | Build the **Flask** backend with API endpoints for user authentication and file management operations. |
| **Implement User Authentication** | Allow users to **register, log in, log out**, and securely protect user accounts through authentication. |
| **Implement File Management Features** | Enable authenticated users to **upload, view, download, and delete files**. Store file metadata in the database and the uploaded files locally during development. |
